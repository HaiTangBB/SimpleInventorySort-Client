# 使用说明 (Usage Instructions)

## 1. 安装步骤
1.  确保您已经安装了 **Minecraft 1.21.11-pre3** 版本。
2.  安装对应版本的 **Fabric Loader**。
3.  将本模组文件 `test-inv-1.0.0.jar` 放入游戏目录下的 `mods` 文件夹中。
4.  启动游戏。

## 2. 功能介绍
本模组提供纯客户端的背包整理功能。

### 2.1 触发方式
- **GUI 按钮**: 打开任意容器（背包、箱子等），在界面右上角会显示一个 **"S"** 按钮。点击该按钮即可触发整理。
- **快捷键**: (当前版本暂禁用，请使用 GUI 按钮)

### 2.2 整理规则
- **保留快捷栏**: 整理操作**不会**影响快捷栏（最下方一排，Slot 0-8）的物品。
- **排序顺序**: 物品将按照以下优先级排序：
    1.  工具/装备 (Tools/Armor)
    2.  方块 (Blocks)
    3.  食物 (Food)
    4.  其他杂项 (Misc)
    - 同类物品按 ID 字母顺序排列。

### 2.3 配置文件
配置文件位于 `.minecraft/config/sortinventory-client-1.21.11pre3.json`。
- `showGuiButton`: 是否显示界面上的 "S" 按钮（默认: true）。

## 3. 注意事项
- 本模组为**纯客户端**模组，无需服务器安装即可使用。
- 为防止被服务器误判，整理操作有 **0.5秒** 的冷却时间。
- 在网络延迟较高的服务器上使用时，请等待整理操作完全停止后再关闭背包，以防物品位置同步失败。
